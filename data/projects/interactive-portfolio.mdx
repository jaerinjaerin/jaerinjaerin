## 프로젝트 상세

### 🎯 프로젝트 목표

단순히 정보를 나열하는 포트폴리오가 아닌, **방문자가 경험하고 기억하는 포트폴리오**를 만드는 것이 목표였습니다.

### 🚀 주요 기능

#### 1. 3D 인터랙티브 배경

- Three.js를 사용한 파티클 시스템
- 마우스 움직임에 반응하는 3D 구체
- 부드러운 카메라 이동과 회전

#### 2. 스크롤 기반 애니메이션

- Framer Motion을 활용한 섹션 전환
- IntersectionObserver를 통한 뷰포트 감지
- 스태거드 애니메이션으로 요소들이 순차적으로 나타남

#### 3. 프로젝트 상세 페이지

- MDX 기반 동적 콘텐츠
- 이미지 갤러리와 라이트박스
- 코드 스니펫 하이라이팅

### 💡 기술적 도전과 해결

#### 성능 최적화

Three.js를 사용하면서 겪은 가장 큰 도전은 **성능 최적화**였습니다.

```typescript
// 동적 임포트로 초기 로딩 최적화
const StarCanvas = dynamic(() => import('@/components/star-canvas'), {
  ssr: false,
  loading: () => <div>Loading...</div>,
});

// useFrame 내부에서 최적화
useFrame((state, delta) => {
  if (ref.current) {
    ref.current.rotation.x -= delta / 10;
    ref.current.rotation.y -= delta / 15;
  }
});
```

#### 반응형 3D 경험

다양한 화면 크기에서 일관된 3D 경험을 제공하기 위해:

```typescript
const [dpr, setDpr] = useState(1);

useEffect(() => {
  // 모바일에서는 pixel ratio를 낮춰 성능 향상
  const pixelRatio = window.devicePixelRatio;
  setDpr(pixelRatio > 2 ? 2 : pixelRatio);
}, []);
```
